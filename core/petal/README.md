# Petal

## Usage

```sh
yarn add --dev @flowr/petal
```

Add the scripts and commit hooks to your package.json:

```json
{
    "scripts": {
        "test": "petal test",
        "lint": "petal lint",
        "build": "petal build",
        "commit": "petal commit",
        "release": "petal release"
    }
}
```
